今回の課題でフィボナッチ数列のプログラムを作りました。はじめは簡単だと思っていましたが、
意外と難しいところもありました。
特に困ったのは、最初にずっと「0」だけが出てしまったことです。
これは最初の２つの数字を正しくセットしていなかったのと、計算のループをちゃんと確認していなかったからです。
コードを見直して、seq[0] = 0 と seq[1] = 1 を入れ、計算の仕方を直したら正しく動くようになりました。

プログラムは複数のクラスに分けました。Fibonacci で数列を計算して、Printer は出力だけを担当し、
Main は入力処理と繰り返しを管理しています。この分け方でわかりやすくなったと思います。

繰り返しは while と for の２つを使いました。while で入力を繰り返し受けて、
for で数列を計算しました。条件にも合っています。

この課題で、仕様をよく理解することと、問題が起きた時に落ち着いて原因を探すことが大切だと学びました。

やり方:

1. ターミナルでソースコードのある source code フォルダに移動します。
. cd path/to/file名 (aijus_ワグナースティーブン)

2. 以下のコマンドでコンパイルします。
. javac *.java

3. コンパイル後、以下のコマンドでプログラムを実行します。
. java Main

4.プログラムは N（70以下の整数）を入力するとフィボナッチ数列を表示し、'quit' 'exit' を入力すると終了します。
